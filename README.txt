Для создания изображения требуется изменить config файл, далее запустить код, как только он выполнится, изображение будет лежать в папке Rendered. 
a, b, c, d , e, f, g проходят линию, не рисуя.
+,- поворот налево и направо соответственно.
[ - сохраняет в стэк, ] - возвращает сохраненное состояние из стэка.
В config кроме аксиомы и правил, также настраивается кол-во итераций, угол и будет ли угол случайным (от 0 до выбранного значения угла). 
Любые другие символы используются только для корректировки генерации.
